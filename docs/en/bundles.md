# 📦 Role-Based Skill Bundles

Bundles are collections of skills curated for specific professional roles. Instead of guessing which skills to use, jump into a **Pack** and get started.

---

## 🎨 1. Web Wizard Pack
For designers and frontend developers refocusing on interaction and aesthetics.
*   **Core Skills**: `@frontend-design`, `@ui-skills`, `@animejs-animation`, `@magic-ui-generator`, `@iconsax-library`
*   **Focus**: Achieving extreme visual excellence, micro-animations, and enforcing Design-First scaffolding.

---

## 🛡️ 2. Security Engineer Pack
For auditing codebases, performing red-team scenarios, and identifying vulnerabilities.
*   **Core Skills**: `@security-auditor`, `@differential-review`, `@security-scanning-security-hardening`, `@red-team-tactics`, `@solidity-security`
*   **Focus**: DevSecOps pipelines, PR blast-radius calculation, and MITRE ATT&CK simulations.

---

## 🧞 3. Agent Architect Pack
For engineers building complex AI systems and workflow automations.
*   **Core Skills**: `@agent-orchestrator`, `@autonomous-agent-patterns`, `@agent-tool-builder`, `@agent-evaluation`, `@mcp-builder`
*   **Focus**: Resolving multi-agent interactions, strictly typing tools with JSON schemas, and scaling MCP servers.

---

## 🐍 4. Data & Backend Pro Pack
For heavy-duty data engineering and hyper-performant microservices.
*   **Core Skills**: `@python-pro`, `@fastapi-router-py`, `@pydantic-models-py`, `@database-design`, `@sql-pro`
*   **Focus**: Building scalable REST APIs, handling massive concurrent connections, and deeply optimizing SQL OLAP queries.

---

## 🧪 5. QA & Testing Pack
For ensuring every single commit is production-ready.
*   **Core Skills**: `@testing-qa`, `@e2e-testing`, `@tdd-workflow`, `@webapp-testing`, `@lint-and-validate`
*   **Focus**: Enforcing the TDD Red-Green-Refactor cycle, automating browser interactions via Playwright, and strictly gating PRs with linters.
