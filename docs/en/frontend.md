# 🎨 Frontend & UI Hub: Visual Excellence

This guide focuses on skills that transform basic layouts into premium, interactive web experiences.

---

## 🏗️ Core Frontend Tools

### 1. 🖌️ @frontend-design
A specialized senior frontend designer-engineer persona. Focuses on **distinctive, production-grade aesthetics** far beyond generic UI layouts.
*   📂 **Path**: `~/.gemini/antigravity/skills/frontend-design/`
*   🎯 **Capabilities**:
    *   **DFII Scoring**: Evaluates designs based on Aesthetic Impact, Feasibility, and Performance.
    *   **Aesthetic Discipline**: Employs specific tone directions (e.g., Editorial, Brutalist, Luxury).
    *   **Anti-Patterns Guard**: Actively prevents systemic AI visual tropes (boring fonts, default layouts).
*   💬 **Example Prompt**: `"Design a pricing page using @frontend-design. Use an 'industrial utilitarian' aesthetic, avoiding Inter/Roboto and default generic layouts."`

---

### 2. 🪄 @ui-skills
Provides a strict set of opinionated, evolving constraints to guide agents when building UI.
*   📂 **Path**: `~/.gemini/antigravity/skills/ui-skills/`
*   🛠️ **Capabilities**:
    *   **Structural Guidelines**: Enforces consistent spacings, radiuses, and paddings based on high-end standards.
*   💬 **Example Prompt**: `"Review this dashboard code using @ui-skills and refactor the padding and typographic hierarchy."`

---

### 3. 🌀 @animejs-animation
An advanced JavaScript animation engine skill for complex, high-performance web choreographies.
*   📂 **Path**: `~/.gemini/antigravity/skills/animejs-animation/`
*   ✨ **Capabilities**:
    *   **Complex Timelines**: Sequences deep, staggered animations across multiple DOM elements.
    *   **Advanced Easing**: Uses custom `spring`, `elastic`, and bezier curves over basic linear transitions.
    *   **SVG Morphing**: Drives complex vector illustrations and dynamic path drawings.
*   💬 **Example Prompt**: `"Use @animejs-animation to orchestrate a staggered, spring-eased hero section entrance."`

---

### 4. 🎨 @magic-ui-generator
Leverages Magic by 21st.dev to generate, compare, and integrate immersive UI components.
*   📂 **Path**: `~/.gemini/antigravity/skills/magic-ui-generator/`
*   🌟 **Capabilities**:
    *   **Multiple Variations**: Explores several distinctly premium design branches before coding.
    *   **Premium Component Linking**: Uses patterns from Aceternity, Shadcn, and Magic UI.
*   💬 **Example Prompt**: `"Use @magic-ui-generator to show me 3 different, avant-garde contact form stylings."`

---

### 5. 🏷️ @iconsax-library
An extensive premium icon framework and AI-driven generation skill to avoid stale default icons.
*   📂 **Path**: `~/.gemini/antigravity/skills/iconsax-library/`
*   💎 **Capabilities**:
    *   **Unified Styles**: Enforces exact styling families (Linear, Bold, Two-tone).
    *   **AI Generation**: Generates bespoke SVG icons via Iconsax AI.
*   💬 **Example Prompt**: `"Find a 'Linear' styled analytics icon from @iconsax-library. If one doesn't fit exactly, generate one using Iconsax AI."`

---

## 🌊 Frontend-First Workflow

1.  **Architecture**: `@frontend-design` to map out the component structure.
2.  **Mockup**: `@magic-ui-generator` to iterate on visual styles.
3.  **Refinement**: `@ui-skills` to polish the spacing and typography.
4.  **Interaction**: `@animejs-animation` + `@design-spells` to add "magic" details.
