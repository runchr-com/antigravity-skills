# 🎨 프론트엔드 & UI 허브: 시각적 탁월함

이 가이드는 기본 레이아웃을 프리미엄급 인터랙티브 웹 경험으로 변환하는 스킬들에 집중합니다.

---

## 🏗️ 핵심 프론트엔드 도구

### 1. 🖌️ @frontend-design
단순한 레이아웃 제너레이터가 아닌, **독창적이고 프로덕션 수준의 미학**을 추구하는 시니어 프론트엔드 디자이너-엔지니어입니다.
*   📂 **경로**: `~/.gemini/antigravity/skills/frontend-design/`
*   🎯 **핵심 역량**:
    *   **DFII 스코어링**: 디자인의 심미성, 구현 가능성, 성능을 수치화하여 평가합니다.
    *   **미학 규율(Tone)**: 인더스트리얼, 브루탈리즘, 럭셔리 등의 명확한 디자인 톤앤매너를 지시합니다.
    *   **안티 패턴 차단**: 뻔한 AI 기본 폰트(Inter 등)나 기본 레이아웃을 엄격하게 금지합니다.
*   💬 **프롬프트 예시**: `"@frontend-design 스킬로 요금제 페이지를 만들어줘. 뻔한 레이아웃을 피하고 '인더스트리얼' 디자인 톤을 유지해."`

---

### 2. 🪄 @ui-skills
UI를 구축할 때 에이전트가 흔들리지 않도록 강력하고도 오피니언이 반영된 **디자인 제약 조건(Constraints)**을 제공합니다.
*   📂 **경로**: `~/.gemini/antigravity/skills/ui-skills/`
*   🛠️ **핵심 역량**:
    *   **구조적 일관성 가이드**: 여백(Spacing), 곡률(Radius), 타이포그래피 계층을 하이엔드 기준에 맞춰 통제합니다.
*   💬 **프롬프트 예시**: `"@ui-skills 스킬을 바탕으로 현재 대시보드 컴포넌트의 여백과 글꼴 계층을 정리해 줘."`

---

### 3. 🌀 @animejs-animation
복잡하고 정교한 고성능 웹 애니메이션을 구현하기 위한 심화 JavaScript 애니메이션 엔진 스킬입니다.
*   📂 **경로**: `~/.gemini/antigravity/skills/animejs-animation/`
*   ✨ **핵심 역량**:
    *   **복합 타임라인**: 여러 DOM 요소가 시간차(Stagger)를 두고 등장하는 정교한 시퀀스를 만듭니다.
    *   **고급 이징(Easing)**: 단순한 직선 모션이 아닌, `spring`, `elastic`, 커스텀 베지어 곡선 등을 사용합니다.
    *   **SVG 모핑**: 백터 이미지의 형태를 바꾸거나 선을 드로잉하는 수준 높은 애니메이션을 수행합니다.
*   💬 **프롬프트 예시**: `"@animejs-animation을 사용해서 히어로 섹션 요소들이 시간차를 두고 튀어 오르는(Spring) 애니메이션을 구성해 줘."`

---

### 4. 🎨 @magic-ui-generator
21st.dev의 Magic을 활용해, 프로덕션에 즉시 투입 가능한 혁신적이고 화려한 UI 컴포넌트를 생성하고 비교합니다.
*   📂 **경로**: `~/.gemini/antigravity/skills/magic-ui-generator/`
*   🌟 **핵심 역량**:
    *   **다중 디자인 변형**: 코드를 짜기 전, 최소 3~4개의 전혀 다른 프리미엄 스타일(글래스모피즘 등) 시안을 제시합니다.
    *   **프리미엄 컴포넌트 라이브러리 연동**: Shadcn, Aceternity, Magic UI 등의 강력한 컴포넌트 패턴을 빌드합니다.
*   💬 **프롬프트 예시**: `"@magic-ui-generator 스킬로 연락처 폼(Contact form)의 전혀 다른 프리미엄 스타일 3가지를 제안해 줘."`

---

### 5. 🏷️ @iconsax-library
흔한 기본 아이콘 대신, 최고의 디자인 일관성을 유지할 수 있는 프리미엄 아이콘 프레임워크 접근 권한입니다.
*   📂 **경로**: `~/.gemini/antigravity/skills/iconsax-library/`
*   💎 **핵심 역량**:
    *   **스타일 패밀리 통일**: Linear, Bold, Two-tone 등 한 프로젝트 내에서 아이콘 스타일을 강제 통일하여 퀄리티를 높입니다.
    *   **AI 자체 생성**: 원하는 컨셉이 없을 경우 Iconsax AI를 통해 새로운 아이콘을 SVG로 생성합니다.
*   💬 **프롬프트 예시**: `"@iconsax-library에서 'Linear' 스타일의 분석 아이콘을 찾아줘. 딱 맞는 게 없다면 AI로 생성해 줘."`

---

## 🌊 프론트엔드 우선 워크플로우

1.  **아키텍처 설계**: `@frontend-design`으로 컴포넌트 구조를 설계합니다.
2.  **디자인 목업**: `@magic-ui-generator`로 시각적 스타일을 반복 실험합니다.
3.  **디테일 정교화**: `@ui-skills`로 간격과 타이포그래피를 세밀하게 조정합니다.
4.  **인터랙션 추가**: `@animejs-animation` + `@design-spells`로 "매직" 디테일을 완성합니다.
