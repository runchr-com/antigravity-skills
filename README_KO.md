# 🚀 Antigravity Awesome Skills Hub (한글 가이드)

[Antigravity Awesome Skills](https://github.com/sickn33/antigravity-awesome-skills) 라이브러리의 *가이드 북**에 오신 것을 환영합니다. 1,300개가 넘는 방대한 AI 스킬과 플레이북 중에서 길을 잃지 않도록, 각 프로젝트 단계(**기획, 디자인, 개발부터 보안, 테스트까지**)에 맞춰 꼭 필요한 핵심 스킬들의 사용 목적과 구체적인 프롬프트 예시를 명확하게 설명하고 안내해 드리는 지침서입니다.

## 🗂️ 스킬 카테고리 (Skill Categories)

프로젝트의 각 단계에 맞는 스킬을 살펴보세요. 카테고리를 클릭하면 상세 가이드와 프롬프트 예시를 볼 수 있습니다.

### 📓 [기획 및 전략 (Planning)](./docs/ko/planning.md)
아이디어를 실행 가능한 계획으로 구체화합니다.
*   `@brainstorming`: MVP 아키텍트 - ROI 분석 및 기능 우선순위 지정.
*   `@product-manager`: 프로덕트 전략가 - 유저 페르소나 및 기능 명세서(PRD) 작성.
*   `@startup-mvp-launcher`: 실행 전략 - 린 캔버스 및 기술 스택 선정.

### 🎨 [프론트엔드 (Frontend)](./docs/ko/frontend.md)
매력적이고 사용자 친화적인 UI/UX를 개발합니다.
*   `@frontend-design`: 모던 웹 디자인 트렌드가 반영된 코드 아키텍처 구성.
*   `@ui-skills`: 컴포넌트 단위의 빠른 레이아웃 스캐폴딩.

### 🐍 [백엔드 (Backend)](./docs/ko/backend.md)
견고하고 확장 가능한 서버 사이드 로직과 데이터베이스를 구축합니다.
*   `@python-pro`: 모던 파이썬 패턴 및 아키텍처.
*   `@database-design`: 스키마 설계 및 데이터베이스 원칙.

### 🛡️ [보안 (Security)](./docs/ko/security.md)
취약점을 방어하고 강력한 보안 시스템을 구축합니다.
*   `@security-auditor`: 코드 보안 감사 및 취약점 스캔.
*   `@red-team-tactics`: MITRE ATT&CK 기반의 레드팀 공격 및 방어 전술.

### 🧞 [에이전트 (Agents)](./docs/ko/agents.md)
자율형 AI 에이전트를 구축하고 효과적으로 조율합니다.
*   `@agent-orchestrator`: 복잡한 작업을 위한 다중 에이전트 조정.
*   `@autonomous-agent-patterns`: 자율형 에이전트 행동 패턴.

### 🧪 [QA & 테스트 (QA Testing)](./docs/ko/qa_testing.md)
자동화된 테스트로 코드의 품질과 안정성을 보장합니다.
*   `@tdd-workflow`: TDD(테스트 주도 개발) 방법론 워크플로우 적용.
*   `@e2e-testing`: Playwright를 활용한 엔드투엔드(E2E) 테스트.

### 🌊 [워크플로우 (Workflows)](./docs/ko/workflows.md)
작업 파이프라인을 자동화하여 생산성을 극대화합니다.
*   `SaaS MVP 출시`: 아이디어부터 출시까지의 2주 완성 워크플로우.
*   `프로덕트 보안 감사`: 전체 코드베이스 보안 스캔 워크플로우.

### 📦 [스킬 번들 (Bundles)](./docs/ko/bundles.md)
특정 목표를 단번에 달성하기 위한 스킬 조합 세트입니다.
*   `@saas-startup-bundle`: SaaS MVP를 2주 안에 출시하기 위한 모든 스킬 모음.
